docknv - Docker with environments
=================================

    One entry-point, multiple deployments


Installation
------------

Quick install:

.. code-block:: bash
 :name: Quick install

 git clone https://bitbucket.org/sharingcloud/docknv.git
 pip install --upgrade -e ./docknv # or sudo
